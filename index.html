Data Privacy

Data stays local inside the browser) and will not leave the client device

Faster inference

Inference happens client side which makes it faster and removes reliabiltiy on the speed of the client's network

Lower costs

Processing data client side will reduce the amount of data that has to be processed and stored server side (usually in the Cloud)

Multi device
